unarchive

(unarchive src)(unarchive src dest)

Unpacks archive entries in file at src into dest directory.

This handles archives, multiple files represented as a single file, for example, a tar file.

uncompress

(uncompress src)(uncompress src dest)

Applies decompression function to file at src into dest file.

This handles compressed files (e.g. gz, xz, bz2) but not archived files (e.g. tar, cpio).
